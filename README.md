#VeriFact – AI-Based Deepfake Detection System

Brief Description
VeriFact is an AI-powered Deepfake Detection System developed to identify manipulated or AI-generated fake images and videos. The project uses Artificial Intelligence and Machine Learning techniques to analyze digital media and determine whether the uploaded content is authentic or deepfake.
The system helps users verify media originality and reduce the spread of misleading or harmful fake content online. Users can upload images or videos through a simple interface and receive instant detection results.

Installation / Execution Steps to Run the Project
Step 1: Clone the Repository
git clone https://github.com/Mahimngeete/VeriFact.git
Step 2: Navigate to the Project Folder
cd VeriFact
Backend Setup (Flask)
Step 3: Install Backend Dependencies
pip install -r requirements.txt
Step 4: Run the Flask Backend Server
python app.py
The backend server will start at:
http://127.0.0.1:5000/
Frontend Setup (React)
Step 5: Navigate to Frontend Folder
cd frontend
Step 6: Install Frontend Dependencies
npm install
Step 7: Start the React Frontend
npm start
The frontend application will start at:
http://localhost:3000/
Working of the Project
1.	User uploads an image or video 
2.	The system preprocesses the media content 
3.	AI models analyze facial patterns and inconsistencies 
4.	Machine Learning algorithms detect manipulation signs 
5.	The system predicts whether the media is Real or Deepfake 
6.	Result is displayed to the user instantly
